# 🍞 Sir SourDough Baking Co.

Marketing and ordering site for Sir SourDough Baking Co. — an artisan sourdough bakery based in Nashua, NH.

**Live site:** [sirsourdough.com](http://sirsourdough.com)

## About

Sir SourDough Baking Co. is a home artisan bakery specializing in naturally leavened sourdough breads and baked goods. This site handles the public-facing brand presence and customer ordering.

## Features

- Mobile-responsive layout with sticky hero image scroll behavior
- Photo gallery showcasing fresh bakes
- Integrated order form (Google Forms embed)
- Lightweight — pure HTML/CSS/JS, no frameworks, fast load times
- Custom domain via GitHub Pages + CNAME

## Stack

| | |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 (custom, no frameworks) |
| Interactivity | Vanilla JavaScript |
| Hosting | GitHub Pages |
| Domain | sirsourdough.com |

## Running Locally

No build step needed — just open `index.html` in your browser, or serve it with any static file server:

```bash
# With Python
python -m http.server 8080

# With Node
npx serve .
```

## Project Notes

This is a real production site for an active small business. Design decisions prioritized fast load times, mobile usability, and ease of updating content without a CMS.
